This project implements a Q-learning based reinforcement learning agent to play simplified Go on a 5x5 board.
It includes:
AI Agent (Agent): Learns strategies using Q-learning.
Random Player (RandomPlayer): Provides a baseline opponent for training and evaluation.
Board Logic (Board): Implements Go rules such as legal moves, captures, and winner determination.
Game Controller (Go): Manages gameplay flow, win/loss checks, and experience replay.
Graphical User Interface (GoUI): Built with Tkinter, allowing interactive play against the AI.
